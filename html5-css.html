HTML5 e CSS

HTML5

HTML5 (Hypertext Markup Language) é uma linguagem de marcaçãoi para a World Wide Web e é uma tecnologia chave da Internet, originalmente proposto por Opera Software. É a quinta versão da linguagem HTML.

Elementos semânticos HTML5 Foram utilizados Descrição
header ✅ Usei para meter o nome da cidade e uma imagem desta em todas as páginas
nav ✅ Usei para meter o menu de navegação com links para as páginas
main ✅ Usei para meter o conteúdo das páginas
article ❌ Não usei
section ✅ Usei, algumas vezes pelo código, para separar a informação por partes

CSS

CCS (Cascading Style Sheet) é um conjunto de regras que determinam como o browser deve apresentar o conteúdo, através deste é possível separar o conteúdo do estilo.

Tipos de seletores do CSS Foram utilizados Descrição
Universal ✅ Este aplica o estilo a todos os elementos da página.
Tipo ✅ Utiliza o nome do marcador a estilizar
Classe ✅ Associa um nome a um estilo
Id ✅ Permite identificar univocamente um elemento do HTML, sendo muito útil no JavaScript
Atributo ✅ Aplica-se a atributos do elemento
Pseudo-classe ✅ Está associado ao estado de um elemento
Pseudo-elemento ✅ Regras que se aplicam a partes de elementos
